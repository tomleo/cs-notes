Arduino¶

Migrating Libraries to 1.0¶

Case 1
Change from
#include “wiring.h” #include “WProgram.h” #include “WConstants.h” #include “pins_arduino.h”
to
#include “Arduino.h”
Case 2
Change from
#include “WProgram.h”
to
#if ARDUINO >= 100
#include “Arduino.h”
#else
#include “WProgram.h”

#endif
